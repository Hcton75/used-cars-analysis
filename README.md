# used-cars-analysis

## Overview
Analysis of 9,500+ used car listings to understand pricing patterns.

## Dataset
- Source: Kaggle
- Size: 9,535 rows (after cleaning)
- Features: Brand, Model, Year, Price, KM Driven, etc.

## Key Findings
1. Negative correlation between km driven and price
2. Automatic transmission cars command higher prices
3. Petrol cars tend to have higher ask prices
4. Top brands: Maruti Suzuki, Hyundai, Honda

- pandas, numpy, matplotlib, seaborn
